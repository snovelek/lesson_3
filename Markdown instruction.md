## Списки

Чтобы добавить ненумерованные списки, необходмо пункты выделить звёздочкой (*). Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например, вот так:
1. Первый пунки
2. Второй пункт
